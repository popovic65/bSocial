# --BSOCIAL--

## Requirements
- Node.js: v18.16.0
- Docker Compose: Installed and configured

## Start Docker Compose:
```bash
docker-compose build
docker-compose up -d
```

## Postman
- Before testing, ensure you have Postman installed.
- Set environment variable to:
  - url: `localhost:3000`

## Tested on
- Ubuntu 23.10
